
# DonateKing

**DonateKing** is a web application that facilitates the donation process by allowing users to donate items to NGOs.

## Project Overview

"DonateKing" is a web-based platform built to facilitate the donation process. Users can easily donate items to NGOs, helping those in need and contributing to various social causes. The project is built using React.js and Tailwind CSS for the frontend and Node.js, Express.js, and MongoDB for the backend.
